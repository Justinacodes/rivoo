Design System Overview

This design system serves two distinct user groups with specialized interfaces:

User Portal

Designed for individuals in high-stress emergency situations. Features calming colors, high contrast, simple navigation, and reassuring design elements to reduce anxiety during critical moments.

Hospital Portal

Designed for medical professionals requiring rapid decision-making. Features data-dense layouts, clinical efficiency, and prioritized information hierarchy for emergency response coordination.
